# Hero video

Drop the AI-handshake clip here as:

- `ai-handshake.mp4`  (required — H.264 MP4, ideally ≤ 1080p, trimmed/looped, muted)
- `ai-handshake.jpg`  (optional poster frame shown before the video loads)

The hero auto-detects the file. Until `ai-handshake.mp4` exists, a placeholder
shows in its place. Source must be properly licensed (e.g. your Vecteezy download).
